## 基于 umi + React Hooks 的go-zone

> Author: Fairy & Chisfee

### 项目简介

项目主要使用React Hooks + dva + umijs Hooks + antd@4构建

### 项目结构

+ 主目录

目录 | 说明
 :-: | :-:
/assets|静态文件
/core|日志配置文件、以及一些常用的公共文件
/common|菜单数据、枚举变量存储
/components|公共组件
/layouts|布局
/models|dva数据管理
/pages|页面
/utils|工具包
app.js|项目入口
global.less|全局样式

### 项目运行 
```
  npm install(依赖包更新时)
  npm start
```
